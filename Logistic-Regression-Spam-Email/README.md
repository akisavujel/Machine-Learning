# 📧 Spam Email Detection 🚫📨

**Spam Email Detection using Logistic Regression & TF-IDF Vectorization** 😎

## 🌟 Project Overview
This project classifies emails as **Spam** (⚠️ unwanted/malicious) or **Ham** (✅ legitimate).  
It uses **Logistic Regression** with **TF-IDF vectorization** for email text processing.

Steps included:  
✨ Data cleaning & preprocessing  
✨ Text vectorization using TF-IDF  
✨ Model training & evaluation  
✨ Predicting new emails as spam or ham  
✨ Visualizing dataset distribution & metrics  

## 📂 Dataset
The dataset contains **1000 emails** with columns:  
📌 `text` → Email content  
📌 `label` → `spam` or `ham`  

Dataset link: [Download CSV](https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/spam_email_dataset_1000.csv)  

## 🛠️ Features
- Preprocessed email text (lowercasing, punctuation removal, stopword removal)  
- TF-IDF vectorized features
