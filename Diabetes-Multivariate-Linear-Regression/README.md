# 🩺 Diabetes Multivariate Linear Regression

## 📖 Project Overview
This project focuses on **predicting diabetes progression** using **multivariate linear regression**. Multiple features such as age, BMI, blood pressure, and more are used to build a regression model that predicts the target variable. The goal is to understand the relationships between features and the target, explore data preprocessing, and evaluate model performance.

## 💻 Technology Used
- 🐍 Python  
- 📊 Pandas – data manipulation  
- 🔢 NumPy – numerical computations  
- 📈 Matplotlib & Seaborn – data visualization  
- 🤖 Scikit-learn – machine learning (linear regression, model evaluation)

## 📂 Dataset
The dataset is available on Kaggle:  
[Diabetes for Multivariate Linear Regression](https://www.kaggle.com/datasets/akisavujel/diabetes-for-multivariate-linear-regression)  

⚠️ Please download the dataset and upload it to your environment before running the notebook.
